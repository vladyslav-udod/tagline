# Tagline

## Features

- Create, edit, and remove taglines
- Drag & drop reordering (DnD-kit)
- Style customization: Variant, Size, Radius, Alignment

---

### Installation

```bash
npm install
```

### Running the App Locally

```bash
npm run dev
```

---

## Tech Stack

- React
- TypeScript
- Vite
- MobX
- Styled-components
- @dnd-kit (drag and drop)
